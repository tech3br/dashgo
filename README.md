### Comandos

*Iniciar servidor em Modo de Desenvolvimento:*

```bash
#!/bin/bash
yarn dev
```

*Criar build:*

```bash
#!/bin/bash
yarn build
```

*Iniciar os testes - TDD:*

```bash
#!/bin/bash
yarn test
```

*Iniciar os testes com assistido - TDD:*

```bash
#!/bin/bash
yarn test --watch
```
